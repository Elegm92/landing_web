# Mundo Mascotas Web

## Tecnologías Utilizadas

Este proyecto utiliza las siguientes tecnologías:

- **HTML5**: Para la estructura del contenido y la organización de las distintas secciones de la página.
- **CSS3**: Para el diseño y los estilos visuales, incluyendo variables CSS y media queries para adaptar la página a diferentes tamaños de pantalla.
- **Flexbox**: Utilizado para organizar algunos elementos de la página, como las imágenes y la galería.

## Características

- Diseño sencillo y limpio.
- Navegación mediante menú superior.

- Formulario de contacto básico.
